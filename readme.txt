билд расположен в папке ./out/artifacts/TextAnalyzer_jar
запуск приложения : java -jar TextAnalyzer.jar inFile outFile stopWordsFile
где:
inFile - файл с исходным текстом
outFile - файл, в который должен быть записан результат
stopWordsFile - список файлов которые программа будет игнорировать при анализе.
